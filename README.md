# OperationCHAD

HTML5 Demo: https://kbjwes77.github.io/gamedev/operationCHAD/

**Keyboard Controls**
 * Arrow Keys - Move/Aim
 * 'X' - Jump
 * 'Z' - Shoot
 
 **Gamepad Controls**
 * D-Pad - Move/Aim
 * A Button - Jump [X Button on Playstation]
 * X Button - Shoot [Square Button on Playstation]
